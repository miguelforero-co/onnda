De src-tauri/src/vad.rs:
```rust
/// Returns (start, end) sample indices of the voiced region in 16kHz mono audio.
/// Applies a 200ms keep-margin (MARGIN = 3200 samples) around detected speech frames.
/// Falls back to (0, len) if the audio is too short to process (< 1 frame).
/// FRAME = 320 samples (20ms at 16kHz).
pub fn vad_trim(samples: &[f32]) -> (usize, usize);
```
Constantes internas: `FRAME = 320`, `MARGIN = 3200`. `n_frames = samples.len() / FRAME`.
Contrato observable:
- Si `samples.len() < FRAME` (n_frames == 0) → retorna `(0, samples.len())` (rama de fallback explícita).
- `start` nunca es negativo (usa `saturating_sub(MARGIN)`), así que para audio sin voz detectada o corto, `start == 0`.
- `end` nunca excede `samples.len()` (usa `.min(samples.len())`).
- En audio totalmente silencioso (todos los frames no-voz), `first_voiced`/`last_voiced` son None → `first = 0`, `last = n_frames-1` → start saturado a 0, end clamped a len.
