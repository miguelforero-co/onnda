onnda — Configurador por tabs

Cada tab (Speaking / Transcribing) guarda su propia config completa — paleta, forma, color y posición vertical. Cambia de tab sin perder lo del otro. Activa Notch físico para ver qué tapa el hardware del MacBook y baja la Posición vertical si la onda se pierde arriba. Copiar JSON exporta los DOS al tiempo.
